# Contacts App

## Overview

This is a simple JavaScript application that displays a list of contacts using **lit-html** for rendering. Each contact is shown as a card with a name and a button that allows the user to view additional information.

## Features

* Displays a list of contacts
* Toggle button to show or hide contact details
* Shows phone number and email for each contact
* Dynamic rendering using **lit-html**

## Technologies Used

* JavaScript (ES6)
* lit-html
* HTML & CSS

## How It Works

The application loads a list of contacts from `contacts.js` and renders them as cards on the page.
Each card contains a **Details** button. When the button is clicked, the application toggles the visibility of additional information such as the contact's phone number and email.

## Run the Project

1. Clone the repository
2. Open `index.html` in your browser
